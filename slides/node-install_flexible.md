### NodeJS / instalación flexible
<hr/>

* **Node Version Manager** (nvm)
* Windows: `nvm-windows` instalador en [github](https://github.com/creationix/nvm)
* Ubuntu: `nvm` en [github](https://github.com/creationix/nvm)
    ```shell
    #instalación manual
    curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.31.0/install.sh | bash
    command -v nvm
    ```
